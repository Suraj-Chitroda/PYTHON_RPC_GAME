# PYTHON_RPS_GAME
This is a very basic console based Rock Paper Scissor game build using python


## PLAY

```bash
python r_p_s.py

Welcome to RPS Game using Python

Type(q) to QUIT?
And here you go


Choose
ROCK(r), PAPER(p), SCISSOR(s) : s
s  vs  p
You won

Choose
ROCK(r), PAPER(p), SCISSOR(s) : r
r  vs  p
Huh! you loss

```
